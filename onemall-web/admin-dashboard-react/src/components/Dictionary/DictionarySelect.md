次组件跟使用 Antd extends Select，使用方法跟 Select 一样

## API

### DictionarySelect

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| dicKey    | 字典key值                                 | string[]  | [] |
| defaultValue    | 来自 antd Select组件             | string、number、boolean  | [] |

### Demo
```jsx harmony
  <DictionarySelect dicKey="gender" defaultValue="1" />
```

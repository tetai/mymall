## API

### DescriptionList

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| dicKey    | 字典的key                                 | string  | [] |
| dicValue    | value值                                 | string、number、boolean  | [] |


### Demo
```jsx harmony
  <DictionaryValueText dicKey="gender" dicValue="2"/>
```
